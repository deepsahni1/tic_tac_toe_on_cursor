This directory contains screenshots of the Tic Tac Toe game:

1. game-preview.png - Main game board
2. game-in-progress.png - A game in progress
3. x-wins.png - Shows X winning with celebration effects
4. draw-game.png - Shows a draw game

To take actual screenshots:
1. Run the game using `npm run dev`
2. Open your browser to the local development server
3. Take screenshots at various game states
4. Save them with the names above in this directory
